# 🛒 Top Mall Utilidades — Sistema de Gestão de Estoque

> Sistema web completo desenvolvido para cliente real, para gestão de estoque de loja de utilidades domésticas com vendas em múltiplos marketplaces.

🌐 **[Acesse o sistema online](https://top-mall-utilidades.web.app)**

---

## 📋 Sobre o Projeto

A **Top Mall Utilidades** precisava de um sistema para controlar o estoque de produtos vendidos no Mercado Livre, Shopee, TikTok Shop e Kwai — tudo em um só lugar, com acesso de qualquer dispositivo.

**Versão 1.0** entregue em março de 2026. **Versão 2.0** com integrações reais de marketplaces em desenvolvimento.

---

## ✨ Funcionalidades

- ✅ **Dashboard** — métricas de estoque em tempo real
- ✅ **Gestão de produtos** — cadastro, edição, exclusão com imagem
- ✅ **Multi-plataforma** — produto em várias plataformas simultaneamente
- ✅ **Vendas** — registro com desconto automático de estoque
- ✅ **Pedidos** — fluxo Novo → Separando → Enviado
- ✅ **Etiqueta térmica** — impressão 100×150mm para impressora cnfujun
- ✅ **Cancelamentos** — com motivo e devolução de estoque
- ✅ **Relatórios** — gráficos de vendas por período e canal
- ✅ **Exportação** — Excel, PDF e CSV
- ✅ **Multi-usuário** — perfis Admin e Colaborador
- ✅ **Sincronização em tempo real** — Firebase onSnapshot

## 🏗️ Arquitetura

```
top-mall-utilidades/
├── index.html          # Sistema completo (SPA)
├── firebase.json       # Configuração de hospedagem
├── .firebaserc         # Projeto Firebase
└── 404.html
```

Sistema construído como **Single Page Application (SPA)** — uma única página HTML que gerencia todas as funcionalidades via JavaScript, com banco de dados NoSQL na nuvem.

## 🚀 Como usar

1. Acesse **[top-mall-utilidades.web.app](https://top-mall-utilidades.web.app)**
